# 🚀 快速开始指南

## 安装完成后的使用

### 1️⃣ 启动服务器

```bash
./start.sh
```

等待看到以下信息：

```
==================================================
  服务器信息
==================================================
  访问地址: http://localhost:8000
  API 文档: http://localhost:8000/docs
  健康检查: http://localhost:8000/api/health

  按 Ctrl+C 停止服务器
==================================================
```

### 2️⃣ 打开浏览器

访问: **http://localhost:8000**

### 3️⃣ 上传图片

1. 点击上传区域或拖拽图片
2. 选择一张清晰的照片（建议：风景、物体、室内场景）
3. 点击"生成 3D 场景"按钮

### 4️⃣ 等待处理

- 首次运行会自动下载模型（~2GB，仅一次）
- 处理时间：10-60秒（取决于图片大小和 GPU 性能）
- 可以看到实时进度提示

### 5️⃣ 交互 3D 场景

生成完成后会自动打开 3D 查看器：

- 🖱️ **左键拖拽** - 旋转视角
- 🖱️ **右键拖拽** - 平移场景
- 🖱️ **滚轮** - 缩放

### 6️⃣ 停止服务

在另一个终端运行：

```bash
./stop.sh
```

或在启动的终端按 `Ctrl+C`

## 💡 推荐测试图片

第一次使用建议用以下类型的图片：

- ✅ 风景照片（山、海滩、建筑）
- ✅ 室内场景（房间、办公室）
- ✅ 物体特写（家具、雕塑）
- ✅ 街景照片

**注意**：
- 图片要清晰（避免模糊）
- 光线充足（避免太暗）
- 场景有深度（不要纯平面）
- 文件小于 10MB

## 🔍 常见问题

### 启动失败？

检查虚拟环境：
```bash
source .venv/bin/activate
sharp --help
```

### 处理太慢？

首次运行会下载模型，之后会快很多。Apple Silicon 使用 MPS 加速，已经比 CPU 快很多了。

### 3D 场景不显示？

1. 检查浏览器控制台是否有错误
2. 确认使用现代浏览器（Chrome/Edge/Firefox）
3. 检查 PLY 文件是否生成：`ls backend/uploads/*/output/`

### 端口被占用？

启动脚本会自动清理端口 8000，或手动运行：
```bash
./stop.sh
```
